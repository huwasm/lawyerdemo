Design 1: Tab-Based Layout — Desktop & Tablet

How the tab-switching concept from the mobile mockup adapts to larger screens. On desktop, both panels fit side by side — tabs switch document views in the left panel. On tablet, the layout compresses but maintains the two-panel split.

Desktop 1280 × 800
On desktop, tabs are unnecessary for switching between document and form — both panels fit side by side. Tabs in the left panel switch between the raw document view and a structured summary.
